# Number Guessing Game :

This is my fourth Python project - a simple number guessing game.

The computer selects a random number between 1 to 100 and you have to guess it.

# How to play:-
- Enter a number between 1 to 100  
- The game will tell you if your guess is too high or too low  
- Keep guessing until you find the correct number  
- Type '0' anytime to exit the game  

# Features:
- Random number generation  
- Hint system (higher/lower)  
- Input validation  
- Counts number of attempts  
- Exit option added  
- Simple interactive messages  

# How to run:
python number_guessing_game.py

- Working with random numbers  
- Handling user input
